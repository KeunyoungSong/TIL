# 제목

## 상황(Context)
언제/어디서/왜 이걸 하게 됐는지 2–3줄.

## 문제(Problem)
무슨 문제가 있었는지, 에러/증상을 구체적으로 적기.

## 원인(Cause)
왜 그런지 한 문장으로 요약하고, 근거/맥락을 덧붙이기.

## 해결(Solution)
실제로 한 조치들을 순서대로 적기.
```bash
명령어가 있으면 여기에
```
```python
코드가 있으면 여기에
```

## 배운 점(Takeaways)
다음에 같은 상황이 오면 어떻게 할지, 핵심 교훈 2–4줄.

## 참고(Links)
- 관련 문서/블로그/이슈 링크

## 용어(Glossary)
이번 글에서 헷갈렸던 기술 용어만 2–5개 정리.
- **term**: 간단한 정의
